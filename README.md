HorseFollow Plugin (Hytale)
Project status

In progress

Objective

Create a functional plugin that enables a taming-like mechanic for mounts in Hytale, a feature not yet available in the base game.

What it does

This plugin creates a bond between the player and a mount (horse).

When the player mounts a horse and uses the command:

The mount becomes linked to the player

The horse behavior is modified

If the player moves beyond a defined distance, the horse is pulled back to the player

The mount stays consistently close to its owner

The goal is a simple, functional, and reliable follow system, without complex AI or unnecessary mechanics.
